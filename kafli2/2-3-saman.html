SET og RESET saman

📖Kafli 2.3
⏱️~20 mínútur
🎯Miðlungs

START/STOP rás

Algengasta notkun SET og RESET er að búa til START/STOP rás fyrir mótor eða annað tæki. Þetta er eins og klassískur rofamynstur í iðnaði.

Dæmi: Mótorstýring

Heildarmynd af START/STOP rás:

Rung 1: START (SET)

I0.0
S Q0.0

Rung 2: STOP (RESET)

I0.1
R Q0.0
Prófa START/STOP rás
🔌
MÓTOR Q0.0: STÖÐVAÐUR
Þrífasa mótor

Forgangur: RESET vinnur

Í flestum PLC kerfum hefur RESET forgang yfir SET. Þetta þýðir að ef bæði eru virk á sama tíma, þá vinnur RESET og útgangurinn fer í OFF.

Þetta er mikilvægt öryggisatriði – við viljum alltaf geta stöðvað vélar, jafnvel þótt eitthvað annað sé að reyna að kveikja á þeim.

💡
Góð venja

Settu alltaf RESET línuna eftir SET línuna í forritinu. Þannig hefur RESET alltaf forgang.

📷

Hér kemur skjáskot af SET/RESET rás í EasySoft

set-reset-ras-easysoft.png

Í raunveruleikanum myndir þú tengja:

  • I0.0 = Grænn START takki
  • I0.1 = Rauður STOP takki (eða neyðarstöðvun)
  • Q0.0 = Tengi (contactor) sem kveikir á mótor
