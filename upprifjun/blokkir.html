Blokkayfirlit

📚Tilvísunarhandbók

Grunnblokkir í LADDER

Hér er yfirlit yfir helstu blokkir sem þú hefur lært. Notaðu þetta sem fljótlega uppflettingu þegar þú ert að forrita.

Tengiliðir (Contacts)

──┤ ├──
NO Contact

Normally Open - leiðir þegar inntak er ON

──┤/├──
NC Contact

Normally Closed - leiðir þegar inntak er OFF

Spólur (Coils)

──( )──
Output Coil

Venjuleg spóla - ON meðan lína er virk

──(S)──
SET Coil

Læsir útgang í ON þangað til RESET

──(R)──
RESET Coil

Læsir útgang í OFF þangað til SET

SET vs RESET - Samanburður

Eiginleiki SET RESET
Hlutverk Kveikir og læsir Slekkur og læsir
Áhrif Útgangur → ON Útgangur → OFF
Læsing Helst ON þar til RESET Helst OFF þar til SET
Forgangur Lægri Hærri (vinnur)
Dæmigert notkun START takki STOP / E-Stop
