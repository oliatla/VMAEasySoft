IP tala stillingar

📖 Kafli 1.1
⏱️ ~10 mínútur
🎯 Byrjandi

Inngangur

Áður en við getum byrjað að forrita PLC vélina þurfum við að stilla IP tölu hennar. IP talan er eins og heimilisfang vélarinnar á netinu – hún segir tölvunni hvar hún á að finna vélina.

Í þessum kafla munum við læra hvernig við finnum IP tölu vélarinnar og stillum hana rétt í EasySoft 8.

Finna IP tölu vélarinnar

IP tala vélarinnar er venjulega merkt á límmiða á vélinni sjálfri, eða þú getur fundið hana í gegnum skjáinn á vélinni ef hún er með skjá.

Góð ráð

Skráðu hjá þér IP tölu vélarinnar þegar þú finnur hana – þú munt þurfa hana oft!

Stilla IP tölu í EasySoft

Fylgdu þessum skrefum til að stilla IP tölu í EasySoft 8:

Opnaðu EasySoft 8

Ræstu forritið og búðu til nýtt verkefni eða opnaðu verkefni sem þú ert að vinna í.

Farðu í Communication stillingar

Í efri valmyndinni, veldu Communication → Connection Setup.

Sláðu inn IP tölu

Sláðu inn IP tölu vélarinnar í viðeigandi reit og smelltu á Connect.

📷

Hér kemur skjáskot af Communication stillingar

communication-setup.png

Prófa tengingu

Þegar þú hefur slegið inn IP tölu og smellt á Connect, ætti EasySoft að sýna grænan lit eða „Connected" til að staðfesta að tenging hafi tekist.

⚠️
Athugaðu

Ef tenging tekst ekki, athugaðu hvort tölvan og vélin séu á sama netkerfinu og hvort IP talan sé rétt slegin inn.
