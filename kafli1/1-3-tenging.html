Skref 1: Tengjast IoT netinu

PLC vélarnar eru tengdar við IoT netið okkar með snúru. Þú þarft að tengja tölvuna þína við sama net í gegnum WiFi til að geta átt samskipti við þær.

📶
WiFi upplýsingar fyrir C03

Nafn nets: IoT-C03
Lykilorð: Snjalli03

Opnaðu WiFi stillingar á tölvunni

Smelltu á WiFi táknið í Windows og finndu IoT-C03 netið.

Tengdu við netið

Sláðu inn lykilorðið Snjalli03 og bíddu eftir tengingu.

📷

Hér kemur skjáskot af WiFi tengingu við IoT-C03

wifi-iot-c03.png

Skref 2: Finna PLC vélina í EasySoft

Nú þegar þú ert á sama neti og PLC vélin þarftu að finna hana í EasySoft.

Veldu Communication flipann

Neðst í EasySoft glugganum eru flipar. Smelltu á Communication flipann.

Smelltu á "Search for devices"

EasySoft leitar að öllum PLC vélum sem eru á netinu. Þetta getur tekið nokkrar sekúndur.

Veldu þína vél

Listi birtist af öllum vélum sem fundust. Veldu vélina sem þú ætlar að vinna með.

📷

Hér kemur skjáskot af Communication flipanum og Search for devices

communication-search-devices.png
ℹ️
Finnst vélin ekki?

Athugaðu að þú sért tengd/ur við IoT-C03 netið. Ef þú ert á öðru neti (t.d. skólanetinu) finnur EasySoft ekki vélina.

Skref 3: Senda forrit á vélina (Download)

Til að senda forritið þitt yfir á PLC vélina þarftu að nota Program / Configuration flipann.

Opnaðu Program / Configuration flipann

Smelltu á Program / Configuration flipann í EasySoft.

Veldu PC ⇒ Device

Þetta sendir forritið úr tölvunni (PC) yfir á PLC vélina (Device).

📷

Hér kemur skjáskot af Program / Configuration flipanum og PC ⇒ Device

program-configuration-pc-device.png

Skref 4: Password Protection gluggi

Þegar þú sendir forrit kemur gluggi sem spyr um lykilorðsvörn. Við notum það ekki.

Veldu "No" við Password Protect Program

Við viljum ekki setja lykilorð á forritið okkar.

Hakaðu í "Don't show this message again"

Þá þarftu ekki að sjá þennan glugga í hvert skipti sem þú sendir forrit.

Smelltu á OK

Forritið er nú sent yfir á vélina!

📷

Hér kemur skjáskot af Password Protect glugganum

password-protect-no.png
⚠️
Athugið: Vélin fer í STOP

Þegar þú sendir nýtt forrit á vélina fer hún sjálfkrafa í STOP ham. Þú þarft að setja hana aftur í RUN til að forritið keyri.

Skref 5: Setja í RUN ham

Eftir download þarftu að setja PLC vélina í RUN ham til að forritið keyri.

🔴
STOP

Forrit keyrir ekki, útgangar eru OFF

🟢
RUN

Forrit keyrir, útgangar virka

💡
Hvernig setja í RUN?

Þú getur breytt á milli RUN og STOP í Communication flipanum, eða beint á PLC vélinni ef hún er með skjá eða rofa.

Skref 6: Fara í Online ham

Online hamur gerir þér kleift að sjá hvað er að gerast í forritinu í rauntíma. Þetta er mjög gagnlegt til að villuleita og skilja hvernig forritið virkar.

Farðu í Communication flipann

Smelltu á Communication flipann neðst í glugganum.

Smelltu á Online

EasySoft tengist við vélina og byrjar að sýna rauntíma gögn.

📷

Hér kemur skjáskot af Communication flipanum og Online hnappnum

communication-online.png
Hvað sérðu í Online ham?
🟢 Virkir tengiliðir

Tengiliðir sem eru TRUE lýsast upp í grænu

🔵 Virkar spólur

Spólur sem eru ON sýna stöðu sína

🔢 Gildi

Teljara- og tímagildi uppfærast í rauntíma

⚡ Rafstraumur

Þú sérð hvar "rafstraumur" flæðir í LADDER

📷

Hér kemur skjáskot af Online ham - LADDER með rauntíma gögnum

⚠️
"Program mismatch" við Online

Forritið í EasySoft er ekki það sama og í vélinni. Sendu forritið aftur með PC ⇒ Device.

⚠️
Útgangar virka ekki eftir Download

Þú gleymdir líklega að setja vélina í RUN ham!
