Flýtileiðir

  • i1-i8 - Bæta við input snertum
  • M1-M16 - Bæta við marker (snerta eða spóla)
  • Q1-Q4 - Bæta við output spólu
  • T1-T8 - Bæta við tíma
  • N - Nýtt network
  • ←→↑↓ - Færa cursor
  • Space - Skipta um tegund (NO↔NC, Coil↔SET↔RST)
  • + - Bæta við parallel branch
  • Delete - Eyða íhlut
